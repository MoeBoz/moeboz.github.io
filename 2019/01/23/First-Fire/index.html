First-Fire

Bozilla
01-23-2019
nichijou

第一总是十分难产。一旦开始，被喜欢的人绑架起来，也许就会停不下来了。

起因

终于开始写自己的第一篇了，总想着一切都准备妥当了再去做，可是故事就是因为猝不及防的发生才值得被书写下来。成为黑历史也是自我成长历史的一部分，想想还是蛮开心的。

最开始是想为自己设计的sofubi建立一个可以贩售用的网站，可是连玩具原型都没有想出来，搭建网站这个事情就一直也没有提上日程。直到我进入了草莓县，草莓县人均码农，人均美少女，每天都在激励我一定要做点什么才能与县民相称，能有话题和大家聊。可是每次自说自话时，总是与tl内容格格不入……最初是看到shioko每天都会说一点关于搭建博客的事情，还挺感兴趣的，就想着，她说的这么容易真的可以嘛，心里默默想了一个多月居然没有动手开始弄，为自己的懒癌点个bad。

搭建

就在一周前，和小蓝打了个赌，在20号之前自己的博客必须要上线。当然，那个时候并没有说明一定要有什么内容就算是搭出来有个框架，弄明白自己在做什么就好了。于是就开始研究怎么搭建。

正如shioko说的，搭建起来真的很容易，可是问题接踵而至。比如，一开始我想多放图片，各种图片的排布就很让人头疼；还有就是MD的渲染，MD has so many different flavors，果然自己重写一个才是硬道理TUT。对于我这种还在北方不死院徘徊的不死人，需要更多的积累来拔旗。

随后在草莓县结识了kiyoko，也是本站第一个友链 懒云休憩室 的持有者。能有朋友一起遇到问题解决问题真的很开心，之后遇到问题，实在是苦恼我还是会去县里面求教的ww

内容

好久没有动手写过什么东西了，就连写纸都是有想好的框架，和老师讨论过后才开始动手，也算是准备妥当才开始的事情。。。自己想写点什么已经是很久以前的事情了。【就算是观后感，评论之类的都没有写过超过100字。回头看过去全都是“嗷嗷嗷嗷”“啊啊啊啊啊啊啊”和“哈哈哈哈哈哈哈哈哈”，完全智障的表达，也没有把当时的体会写下来，过几年之后像看看是不是有相同的感受，就连一点点的相关记录都没有，现下还是有点后悔。

自己一个人呆着太久，说话都不合逻辑。有时候和别人聊天都会出现表达失误，讲不清故事或是连词连着用，前后语义重复什么的，也许和这种为了脱离人群，最后真的脱离人群，但又开始嫌弃自己丧失和人群之间信息沟通能力的事实有很大的关系。

为了结束这种持续时间有点长的自厌期，又或者是为了给未来的自己有一点成长经验积累的参考才决定开始写自己的博客。原本的计划是放自己的照片和画的小画，仔细看了下画的插画还有拍摄的照片，因为学习的不系统，根本不敢放到太太多的地方求指教。有了这个博客之后，也可以放练习的成果，和过程。也许能通过一点一滴的积累找到适合的方法来完成学习这两项技能。

今年还有几个flag想随着这个博客一起立起来（可是博客的flag已经拔了，若是说是个长期flag，那还是立得高高的TUT），一个是学习日语，还有一个是读20本书。为什么说20本书就是flag……因为平时看漫画多于文字类的我语言匮乏，就连安利别人都找不到合适的形容词，好怕自己今年看了20部漫画都没有看完20本全文字的书……
